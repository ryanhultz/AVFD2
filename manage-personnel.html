Edit the roster, then download personnel.json and upload it to your GitHub repo.

How to update the roster

  1. Edit the table below directly — or — paste/import a CSV (see Import section)
  2. Click Download personnel.json
  3. Go to your GitHub repo → click personnel.json → pencil icon (Edit) → paste contents — or — drag-drop the file to upload
  4. Commit the change. The run report app picks it up automatically on next load.

CSV format (for import from Excel / Google Sheets):
Name,Badge,Qual
Smith John,42,emt/int
Qual options: int  |  emt  |  emt/int  |  leave blank
Roster — 0 members Click any cell to edit
